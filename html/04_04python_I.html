Python · Step 19: Functions for slope and intercept

Write a Python program that:
1. defines a function welcome() that prints a greeting
2. defines a function get_points() that asks for two points (x1, y1, x2, y2)
3. defines a function calculate_slope(x1, y1, x2, y2) that calculates and returns m
4. defines a function calculate_intercept(x1, y1, m) that calculates and returns t
5. prints the complete linear function in the form f(x) = y = m * x + t.

Example:
Enter x1: 2
Enter y1: 3
Enter x2: 6
Enter y2: 11
Output → f(x) = y = 2.0 * x - 1.0
