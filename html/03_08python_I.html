Python · Step 8: Check if a year is a leap year

In the Gregorian calendar, a year is a leap year if it follows these rules:

• A year is a leap year if it is divisible by 400.
• A year is also a leap year if it is divisible by 4, except when it is divisible by 100.

Write a Python program that asks the user for a year and then checks whether it is a leap year.

Example:
Enter a year: 2000
Output → 2000 is a leap year.

Enter a year: 1900
Output → 1900 is not a leap year.
