Python ¬∑ Step 4: Calculator with match / case

Write a calculator program.
1. Ask for two numbers.
2. Ask for an operation (+, -, *, /).
3. Use match / case (Python 3.10+) to choose what to do.

Example dialog:
Enter first number: 12
Enter second number: 4
Choose operation (+, -, *, /): /
Output ‚Üí The result is: 3.0

‚ö† In normal Python 3.10+ you can write:

match op:
    case "+":
        result = num1 + num2
    case "-":
        result = num1 - num2
    case "*":
        result = num1 * num2
    case "/":
        if num2 == 0:
            print("Error: division by zero")
        else:
            result = num1 / num2
    case _:
        print("Invalid operation")
